# Circuit Quest — NEC Code Trainer

A game-like study app for the National Electrical Code (NEC), covering the
**2020 / 2023 / 2026** editions. Guided chapter lessons with real-world
examples, level-based tests (Apprentice / Journeyman / Master), medals, stars,
XP and daily streaks, word search, flashcards, term match, timed Code Rush,
spot-the-violation challenges, a searchable glossary, and an Ugly's-style quick
reference. Includes a Journeyman timed exam and an auto-solving Test Bot.

## Run it
Open `index.html` in any modern browser, or visit the GitHub Pages URL once
enabled (Settings → Pages → deploy from the `main` branch, root folder).

On iPhone/iPad: open the Pages URL in Safari → Share → **Add to Home Screen**
for a full-screen, app-like experience. Progress saves on the device.

## Note on content
Circuit Quest is an independent study aid with **original** educational content.
It explains NEC concepts in plain language and cites exact section numbers, but
does **not** reproduce the copyrighted NEC® text (published by NFPA®). Read the
official code free at https://www.nfpa.org/free-access. Always verify against
the adopted edition and local amendments.
